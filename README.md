# CentripetalForce Portal

Welcome to the official repository for **CentripetalForce**, a mobile and web application development studio.

This repository hosts our static landing portal, bilingual user manuals, and the official privacy policy for our flagship application **ZenGoStudio** (a premium Go / Weiqi study and play companion) on GitHub Pages.

## Live Website
The website is served directly via GitHub Pages at your configured GitHub Pages URL (e.g., `https://CentripetalForce.github.io/` or similar).

## Structure
- `index.html` - Home page / Studio Landing introducing ZenGoStudio.
- `docs.html` - Bilingual (Spanish/English) documentation manual for ZenGoStudio rules and game modes.
- `privacy.html` - Official Privacy Policy compliant with Google Play Store guidelines.
- `style.css` - Custom premium theme layout matching the ZenGoStudio application's visual language.
